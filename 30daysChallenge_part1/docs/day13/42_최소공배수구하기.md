## 1. 문제 분석

아래 식을 통해 두 수의 최소 공배수를 구할 수 있다.

```
A와 B의 최소 공배수 = A * B / (A와 B의 최대 공약수)
```

즉, `유클리드 호제법`을 이용해서 문제를 해결할 수 있다.

## 3. 슈도코드 

``` 
gcd(a, b) : 
    if b == 0 : 
        a가 최대 공약수 

    else :
        gcd(작은 수, 큰 수 % 작은 수) 

t : 테스트 케이스 

for t만큼 반복 : 
    a = 첫 번째 수 
    b = 두 번째 수 

    결과값 = a * b / gcd(a, b)
    print(결과값)

```
